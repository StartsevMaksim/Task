Порядок работы
1. Открыть файл "tape.html"
2. Вам доступны след. функции: 
	1. Фильтрация фотопостов по авторам и описанию в лексеграфическом порядке (выбираем одну из радиокнопок и нажимаем кнопку "Filter")
	2. Залогиниться в приложение. Для этого нажимаем кнопку "Sign in", переходим на страницу авторизации. В первой строке вводим логин = SiteUser
		Во второй строке пароль = 123456. Нажимаем кнопку "Log in"
3. Попали на начальный экран, но уже в статусе авторизированного пользователя. Нам доступны след. функции:
	1. Фильтрация.
	2. Выход из профиля (Нажатием кнопки "Log out").
	3. Добавление нового поста (Кнопка "Add Photo").
	4. Редактирование уже имеющихся фотопостов (Кнопка "Edit Post").
4. Добавление фотопоста: Нажимаем кнопку "Add Photo" и попадаем на экран добавления поста. Следуем пунктам: выбираем фотографию,
				!!!ЗАМЕЧАНИЕ!!! нельзя добавлять фотографию, которая уже есть на сайте
		Заполняем поле описания поста и жмем кнопку "Save"
5. Редактирование постов: Нажимаем кнопку "Edit Post". Есть возможность изменить фотографию и описание для СВОЕГО поста, а так же удалить свой пост
	Для редактирования содержимого выполняем последовательность команд прописанных в первой форме, а именно
		1. Указываем ID поста, который хотим изменить (у авторизированных пользователей ID указан слева от имени автора в блоке информаиции о посте) 
		2. Выбираем фотографию, которой ещё нет на сайте, изменяем описание и фиксируем изменения нажатием клавишы "Save"
	Для удаления поста используем вторую форму, которая находится по странице ниже от первой
		1. Указываем ID поста, который хотим удалить
		2. Удаляем пост нажатием кнопки "Save"
